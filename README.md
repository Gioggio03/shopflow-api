# ShopFlow API

E-commerce / order management REST API built with Spring Boot.
A learning project developed incrementally to cover the core skills of a backend Java developer.

## Tech Stack
- Java 21
- Spring Boot 4.1
- Maven
- PostgreSQL (from Phase 2)

## Getting Started
Run the application:

    ./mvnw spring-boot:run

The app starts on http://localhost:8080
Health check: http://localhost:8080/actuator/health
